## ¡Bienvenido!

Este es un repositorio demo de las capacidades actuales de generación de código con Claude 3.7 y Cursor.
Todo el código generado lo encontrarás en la carpeta *portafolio-app* en su interior está todo el código generado. Nada de lo que está ahí ha sido escrito por un ser humano.

Si quieres hechar un vistazo al prompt utilizado con Cursor, chequea el archivo *prompt.md* en esta misma raiz.
